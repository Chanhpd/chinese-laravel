## Each Vocabulary Entry Includes

- **word**: Chinese characters (simplified)
- **pinyin**: Pronunciation with tones
- **simplified**: Simplified Chinese
- **traditional**: Traditional Chinese
- **part_of_speech**: Word type (noun, verb, adjective, etc.)
- **meaning**: English translation
- **meaning_zh**: Chinese meaning
- **example_sentence**: Example usage in Chinese
- **example_translation**: Example translation in English
- **definition**: Word definition
- **sentences**: Array of 3 quiz sentences with blanks (___) for fill-in exercises
- **level**: HSK1
